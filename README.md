# 1 Предварительная настройка

В командной строке ...\galxe запустить команду: pip install -r /path/to/requirements.txt
В файле ...\galxe\config.py настроить все что необходимо

# 2 Запуск

В командной строке ...\galxe запустить скрипт командой: python .\run.py

```
Config.py
№ строки      Переменная                              Что делает
10            TWITTER_PROFILE_TO_TOKENS               Указывает ссылку на создание файла, где будут храниться данные твиттера и профиля, к которому они привязаны
11            GMAIL_PROFILE_PATH                      Указывает ссылку на создание файла, где будут храниться данные гмаила и профиля, к которому они привязаны
12            DISCORD_PROFILE_TO_TOKENS               Указывает ссылку на создание файла, где будут храниться данные дискорда и профиля, к которому они привязаны
14            GMAIL_DATA_PATH                         Путь к файлу, который содержит в себе данные гмаилов в формате .txt
15            ETH_DATA_PATH                           Путь к файлу, который содержит в себе данные rabby в формате .xlsx
16            PROFILE_DATA_PATH                       Путь к файлу, который содержит в себе данные профилей в формате .xlsx
17            DISCORD_DATA_PATH                       Путь к файлу, который содержит в себе данные дискордов в формате .txt

20            CAMPAIGNS_RESULT_PATH                   Указывает ссылку на создание файла, где будут храниться результаты выполнения квестов и профиля, к которому они привязаны // Пока не работает

28            TOTAL_WINDOWS                           Общее количество окон, которые надо открыть (должно быть не больше max_limit из словаря PROFILE_RANGE)
30            SUBMAIL_DATA                            Данные резервной почты
```
